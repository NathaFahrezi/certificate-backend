# 🛠️ Certificate Backend

Backend untuk aplikasi **Certificate DApp**. Bertugas meng-handle upload sertifikat ke IPFS via Pinata, serta menyimpan hash ke blockchain melalui smart contract Ethereum.

---

## 📦 Tech Stack

- **Node.js** + **Express.js**
- **IPFS via Pinata API**
- **Ethers.js**
- **dotenv**
